  • 參加辦法
    • 活動對象：AU8會員皆可成為AU8娛樂城代理
    • 代理活動內容：
      • 第一層代理回饋0.1%
      • 第二層代理回饋0.05%
      • 第三層代理回饋0.05%
      • 全民代理獲得推廣碼路徑：登入註冊AU8娛樂城→儲值→複製邀請碼/推廣網址或儲存QR CODE
      • 會員可推薦無上限會員數，全民代理回饋亦無上限。
      • 回饋金需要下注量達1倍才可申請提領。
    • 代理紅利方案：
      • 代理可抽下一代會員所產生之紅利（會員的有效投注ｘ紅利％數）
      • 最多可抽到下三代會員紅利（最高0.2％紅利贈點無上限）

      • 領取推薦彩金完成流水後即可提領（計算方式請參閱推薦活動）
